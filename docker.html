Docker Tutorial

How to Use Docker

First, you need to install Docker to your computer. Click here to download docker. Docker supports Ubuntu, Windows, and Mac. Then, you have to pull an Ubuntu image from Dockerhub. To do that you can use the command docker pull ubuntu.

After you exit the Ubuntu container you have to commit a container in order to create a new Docker image; to do this you can use the command docker ps -a , then you have to commit it to the container ID and name the image.

Then, you have to share data between the container and the host. Type in: docker run -it -p an IP address then the host directory followed by /bin /bash. Then you use the vi Editor to create an html file that will be shown on the website you put into the directory.

Lastly, start the apache service using /etc/init.d/apache2 restart. Then on your browser input the IP address and you should see your website.
